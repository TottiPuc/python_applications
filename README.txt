This repository has a great python portfolio.
  
  1. A dictionary => with a large set of data that contains most of the English words, you can query what you want and it will give you        the meaning or meanings of this query. i.e,
     rain:
     ==> Precipitation in the form of liquid water drops with diameters greater than 0.5 millimetres.
     ==> To fall from the clouds in drops of water.

  2. A webMap  => This simple web aplication allows you to easily and quickly get information about the most important volcanoes in the        south part of Colombia.
  
  3. A web Blocker => This simple and clever application block the websites, just passing the name of the web that you want to block,  to      get focus in working hours
